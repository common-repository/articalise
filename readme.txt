For grouping posts together that are all part of one article.

== Description ==
If you are writing something that spans multiple posts (e.g. Guide, FAQ, Product Review), and you want
to group these posts together. You can still use your usual categories and tags.

The article name is prefixed to the post title and a small 'mini menu' is added at the end, or beginning, of the post.

== Installation ==
Just normal like:

1. Upload plugin to the `/wp-content/plugins/` directory
2. Unzip the zip file.
3. Activate the plugin through the 'Plugins' menu in WordPress

Then:

4. Configure where you want it through the 'Settings' -> 'Articalise' menu.
